Salop Web Server - Readme

Este é um arquivo de texto simples para testar o tipo MIME text/plain.

O servidor agora suporta uma ampla variedade de tipos de arquivo:

IMAGENS:
- PNG (Portable Network Graphics)
- JPEG/JPG (Joint Photographic Experts Group)
- GIF (Graphics Interchange Format)  
- ICO (Icon files)
- BMP (Bitmap)
- SVG (Scalable Vector Graphics)
- WebP (Web Picture format)

FONTES:
- WOFF (Web Open Font Format)
- WOFF2 (Web Open Font Format 2.0)
- TTF (TrueType Font)
- OTF (OpenType Font)
- EOT (Embedded OpenType)

MÍDIA:
- MP4 (MPEG-4 Video)
- WebM (Web Media)
- MP3 (MPEG Audio Layer 3)
- WAV (Waveform Audio)
- OGG (Ogg Vorbis)

DOCUMENTOS:
- PDF (Portable Document Format)
- TXT (Plain Text)
- MD (Markdown)

ARQUIVOS:
- ZIP (Compressed Archive)
- GZ (Gzip Compressed)
- JSON (JavaScript Object Notation)
- XML (Extensible Markup Language)

Para testar os tipos MIME, acesse: mime-test.html
